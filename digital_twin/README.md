# Adaptive Dispatch Digital Twin for Surface Mine Truck–Shovel Operations

## Problem

Static truck assignment cannot react to changing queue lengths, travel times, loading times, or equipment failures. This project develops a modular digital twin that continuously observes the system state and adaptively assigns trucks to improve operational performance.

## Project Scope

This repository contains the software implementation of a digital twin for truck–shovel mining operations.

The MVP will include:

- Discrete-event simulation using SimPy
- Multiple dispatching policies
- Adaptive estimated-completion-time dispatch
- Synthetic scenario generation
- KPI calculation
- Experiment runner
- Streamlit dashboard
- Automated testing
- Documentation
